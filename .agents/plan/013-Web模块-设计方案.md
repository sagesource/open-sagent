# 方案：013-Web模块-设计方案

## 1. 背景与目的

### 1.1 背景

Open Sagent框架已完成LLM模块（001-004）、Prompt模块（005）、Memory模块（008、011、012）、Agent核心模块（009）、ReActAgent模块（010）的设计与实现。根据项目系统架构方案 `project_design_web.md`，`open-sagent-web` 模块负责提供WEB-AGENT实现，包括用户系统、对话管理、SSE流式对话交互、Agent版本选择等能力。

当前 `open-sagent-web` 模块仅包含基础pom.xml，后端服务和前端页面均未实现，需要补齐Web模块的完整实现。

### 1.2 目的

1. 实现用户系统：支持邮箱注册、登录、修改个人信息（昵称、密码）
2. 实现对话管理系统：对话列表CRUD、使用SQLite持久化存储
3. 实现Agent版本选择：Sagent-Simple（基于SimpleAgent）和 Sagent-Smart（基于ReActAgent）
4. 实现SSE流式对话接口：支持流式输出、中断、AGENT_ACTION标记透传
5. 实现标题生成Agent：基于SimpleAgent自动生成对话标题
6. 实现前端页面：TypeScript开发，支持暗/亮模式、Markdown渲染、代码块复制
7. 实现全局异常处理：后端异常包装为统一JSON格式返回
8. 生成后端接口SKILL，便于后续开发参考

package ai.sagesource.opensagent.web.entity;

import jakarta.persistence.*;
import lombok.AllArgsConstructor;
import lombok.Builder;
import lombok.Data;
import lombok.NoArgsConstructor;

import java.time.LocalDateTime;

/**
 * 用户实体
 *
 * @author: sage.xue
 * @time: 2026/4/26
 */
@Data
@Entity
@Builder
@NoArgsConstructor
@AllArgsConstructor
@Table(name = "sys_user")
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    /**
     * 邮箱（唯一，作为登录账号）
     */
    @Column(unique = true, nullable = false, length = 128)
    private String email;

    /**
     * 加密后的密码
     */
    @Column(nullable = false, length = 256)
    private String passwordHash;

    /**
     * 对话昵称
     */
    @Column(length = 64)
    private String nickname;

    /**
     * 创建时间
     */
    @Column(nullable = false)
    private LocalDateTime createdAt;

    /**
     * 更新时间
     */
    @Column(nullable = false)
    private LocalDateTime updatedAt;

    @PrePersist
    protected void onCreate() {
        createdAt = LocalDateTime.now();
        updatedAt = LocalDateTime.now();
    }

    @PreUpdate
    protected void onUpdate() {
        updatedAt = LocalDateTime.now();
    }
}

@AllArgsConstructor
@Table(name = "conversation")

     * 创建时间

     * 更新时间

package ai.sagesource.opensagent.web.security;

import io.jsonwebtoken.*;
import io.jsonwebtoken.security.Keys;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;

import javax.crypto.SecretKey;
import java.nio.charset.StandardCharsets;
import java.util.Date;

/**
 * JWT工具类
 *
 * @author: sage.xue
 * @time: 2026/4/26
 */
@Slf4j
@Component
public class JwtUtil {

    @Value("${sagent.jwt.secret}")
    private String secret;

    @Value("${sagent.jwt.expire-hours:24}")
    private int expireHours;

    private SecretKey getKey() {
        return Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    }

    /**
     * 生成JWT Token
     */
    public String generateToken(Long userId, String email) {
        Date now = new Date();
        Date expire = new Date(now.getTime() + expireHours * 60L * 60L * 1000L);

        return Jwts.builder()
                .subject(String.valueOf(userId))
                .claim("email", email)
                .issuedAt(now)
                .expiration(expire)
                .signWith(getKey())
                .compact();
    }

    /**
     * 解析Token获取用户ID
     */
    public Long getUserId(String token) {
        Claims claims = parseToken(token);
        return Long.valueOf(claims.getSubject());
    }

    /**
     * 验证Token是否有效
     */
    public boolean validateToken(String token) {
        try {
            parseToken(token);
            return true;
        } catch (ExpiredJwtException e) {
            log.warn("> JwtUtil | Token已过期 <");
            return false;
        } catch (JwtException e) {
            log.warn("> JwtUtil | Token验证失败: {} <", e.getMessage());
            return false;
        }
    }

    private Claims parseToken(String token) {
        return Jwts.parser()
                .verifyWith(getKey())
                .build()
                .parseSignedClaims(token)
                .getPayload();
    }
}

 * JWT登录态拦截器
 *
 * @author: sage.xue
 * @time: 2026/4/26
 */
@Slf4j
@Component
@RequiredArgsConstructor
public class JwtInterceptor implements HandlerInterceptor {

    private final JwtUtil jwtUtil;

    public static final String HEADER_AUTH = "Authorization";
    public static final String ATTR_USER_ID = "userId";

    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) {
        String token = extractToken(request);
        if (token == null || !jwtUtil.validateToken(token)) {
            response.setStatus(HttpServletResponse.SC_UNAUTHORIZED);
            return false;
        }
        Long userId = jwtUtil.getUserId(token);
        request.setAttribute(ATTR_USER_ID, userId);
        return true;
    }

    private String extractToken(HttpServletRequest request) {
        String bearer = request.getHeader(HEADER_AUTH);
        if (bearer != null && bearer.startsWith("Bearer ")) {
            return bearer.substring(7);
        }
        return null;
    }
}

 * @time: 2026/4/26
 */
@Slf4j
@Service
@RequiredArgsConstructor
public class ChatService {

    private final LLMCompletion llmCompletion;
    private final ConversationService conversationService;
    private final TitleAgentService titleAgentService;

    @Qualifier("simpleAgentConfig")
    private final AgentConfig simpleAgentConfig;

    @Qualifier("smartAgentConfig")
    private final AgentConfig smartAgentConfig;

    private static final PromptTemplate DEFAULT_PROMPT = new PromptTemplate() {
        @Override
        public String render(PromptRenderContext context) {
            return "你是一个 helpful 的AI助手，请尽力回答用户的问题。";
        }
    };

    /**
     * 活跃的SSE连接取消令牌映射
     */
    private final Map<String, CompletionCancelToken> activeTokens = new ConcurrentHashMap<>();

    /**
     * 执行SSE流式对话
     */
    public SseEmitter streamChat(Long userId, String sessionId, String message, String agentVersion) {
        Conversation conversation = conversationService.getBySessionId(sessionId);
        if (!conversation.getUserId().equals(userId)) {
            throw new IllegalArgumentException("无权访问此对话");
        }

        SseEmitter emitter = new SseEmitter(0L); // 无超时
        String emitterId = sessionId + "-" + System.currentTimeMillis();

        // 保存用户消息到前端消息表
        conversationService.saveMessage(conversation.getId(), "user", message);

        // 判断是否需要生成标题
        boolean needTitle = false;
        long messageCount = conversationService.getMessages(conversation.getId()).size();
        if (messageCount <= 5) {
            needTitle = true;
        }

        // 创建Agent和Memory
        Memory memory = new MultipleSQLLiteMemory(
                sessionId,
                "./memory.db",
                50,
                llmCompletion,
                null
        );

        Agent agent = createAgent(agentVersion, memory);

        // 发送标题生成标记（如果需要）
        final boolean finalNeedTitle = needTitle;

        Consumer<StreamChunk> chunkConsumer = chunk -> {
            try {
                if (chunk.getDeltaText() != null) {
                    emitter.send(SseEmitter.event()
                            .name("message")
                            .data(chunk.getDeltaText()));
                }
                if (chunk.getToolCall() != null) {
                    emitter.send(SseEmitter.event()
                            .name("action")
                            .data("AGENT_ACTION[EXECUTE_TOOL]:" + chunk.getToolCall().getName()));
                }
                if (Boolean.TRUE.equals(chunk.getFinished())) {
                    // 流式结束
                    if (finalNeedTitle) {
                        String title = titleAgentService.generateTitle(message, messageCount <= 1);
                        if (!"新对话".equals(title)) {
                            conversationService.updateTitle(userId, conversation.getId(), title);
                        }
                        emitter.send(SseEmitter.event()
                                .name("title")
                                .data(title));
                    }
                    emitter.send(SseEmitter.event()
                            .name("done")
                            .data(""));
                    emitter.complete();
                }
            } catch (IOException e) {
                log.error("> ChatService | SSE发送失败: {} <", e.getMessage());
                emitter.completeWithError(e);
            }
        };

        // 在独立线程中执行流式调用
        new Thread(() -> {
            try {
                CompletionCancelToken token = agent.stream(
                        UserCompletionMessage.of(message),
                        chunkConsumer
                );
                activeTokens.put(emitterId, token);

                // 等待流式完成或中断
                while (!token.isCancelled()) {
                    Thread.sleep(100);
                }
            } catch (Exception e) {
                log.error("> ChatService | 流式对话异常: {} <", e.getMessage());
                try {
                    emitter.send(SseEmitter.event()
                            .name("error")
                            .data(e.getMessage()));
                    emitter.complete();
                } catch (IOException ex) {
                    emitter.completeWithError(ex);
                }
            } finally {
                activeTokens.remove(emitterId);
            }
        }).start();

        // 连接断开时清理
        emitter.onCompletion(() -> activeTokens.remove(emitterId));
        emitter.onTimeout(() -> {
            activeTokens.remove(emitterId);
            emitter.complete();
        });
        emitter.onError((e) -> activeTokens.remove(emitterId));

        return emitter;
    }

    /**
     * 中断对话
     */
    public void cancelChat(String sessionId) {
        activeTokens.forEach((key, token) -> {
            if (key.startsWith(sessionId)) {
                token.cancel();
            }
        });
    }

    private Agent createAgent(String version, Memory memory) {
        if ("smart".equals(version)) {
            return new ReActAgent(
                    "Sagent-Smart",
                    DEFAULT_PROMPT,
                    null,
                    memory,
                    llmCompletion,
                    null, // TODO: 后续接入ToolRegistry
                    smartAgentConfig
            );
        }
        return new SimpleAgent(
                "Sagent-Simple",
                DEFAULT_PROMPT,
                null,
                memory,
                llmCompletion,
                null,
                simpleAgentConfig
        );
    }
}

import { useState, useRef, useCallback } from 'react';
import type { StreamEvent } from '../types';

interface ChatState {
  content: string;
  action: string | null;
  isStreaming: boolean;
  error: string | null;
}

export function useChatStream() {
  const [state, setState] = useState<ChatState>({
    content: '',
    action: null,
    isStreaming: false,
    error: null,
  });
  const eventSourceRef = useRef<EventSource | null>(null);

  const startStream = useCallback((sessionId: string, message: string, agentVersion: string) => {
    // 关闭已有连接
    if (eventSourceRef.current) {
      eventSourceRef.current.close();
    }

    setState({ content: '', action: null, isStreaming: true, error: null });

    const params = new URLSearchParams({ sessionId, message, agentVersion });
    const es = new EventSource(`/api/chat/stream?${params.toString()}`, {
      headers: { Authorization: `Bearer ${localStorage.getItem('token')}` },
    });
    eventSourceRef.current = es;

    return new Promise<{ content: string; title?: string }>((resolve, reject) => {
      let fullContent = '';
      let title: string | undefined;

      es.addEventListener('message', (e) => {
        fullContent += e.data;
        setState(prev => ({ ...prev, content: fullContent }));
      });

      es.addEventListener('action', (e) => {
        setState(prev => ({ ...prev, action: e.data }));
      });

      es.addEventListener('title', (e) => {
        title = e.data;
      });

      es.addEventListener('done', () => {
        es.close();
        setState(prev => ({ ...prev, isStreaming: false }));
        resolve({ content: fullContent, title });
      });

      es.addEventListener('error', (e) => {
        const data = (e as MessageEvent).data || '连接异常';
        es.close();
        setState(prev => ({ ...prev, isStreaming: false, error: data }));
        reject(new Error(data));
      });

      es.onerror = () => {
        es.close();
        setState(prev => ({ ...prev, isStreaming: false, error: '连接已断开' }));
        reject(new Error('连接已断开'));
      };
    });
  }, []);

  const cancelStream = useCallback((sessionId: string) => {
    if (eventSourceRef.current) {
      eventSourceRef.current.close();
    }
    fetch(`/api/chat/${sessionId}/cancel`, { method: 'POST' });
    setState(prev => ({ ...prev, isStreaming: false }));
  }, []);

  return { state, startStream, cancelStream };
}

import React, { useState, useEffect, useRef } from 'react';
import { MessageList } from './MessageList';
import { ChatInput } from './ChatInput';
import { Sidebar } from './Sidebar';
import { ThemeToggle } from './ThemeToggle';
import { useChatStream } from '../hooks/useChatStream';
import type { Conversation, ChatMessage } from '../types';
import * as conversationApi from '../api/conversation';

interface ChatPageProps {
  theme: 'light' | 'dark';
  onToggleTheme: () => void;
}

export const ChatPage: React.FC<ChatPageProps> = ({ theme, onToggleTheme }) => {
  const [conversations, setConversations] = useState<Conversation[]>([]);
  const [currentConv, setCurrentConv] = useState<Conversation | null>(null);
  const [messages, setMessages] = useState<ChatMessage[]>([]);
  const [inputText, setInputText] = useState('');
  const [agentVersion, setAgentVersion] = useState<'simple' | 'smart'>('simple');
  const messagesEndRef = useRef<HTMLDivElement>(null);
  const { state: streamState, startStream, cancelStream } = useChatStream();

  // 加载对话列表
  useEffect(() => {
    conversationApi.list().then(resp => {
      if (resp.code === 0) {
        setConversations(resp.data);
      }
    });
  }, []);

  // 加载当前对话消息
  useEffect(() => {
    if (currentConv) {
      conversationApi.getMessages(currentConv.id).then(resp => {
        if (resp.code === 0) {
          setMessages(resp.data);
        }
      });
    } else {
      setMessages([]);
    }
  }, [currentConv]);

  // 自动滚动到底部
  useEffect(() => {
    messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
  }, [messages, streamState.content]);

  const handleCreateConv = async () => {
    const resp = await conversationApi.create(agentVersion);
    if (resp.code === 0) {
      const newConv = resp.data;
      setConversations(prev => [newConv, ...prev]);
      setCurrentConv(newConv);
    }
  };

  const handleDeleteConv = async (id: number) => {
    const resp = await conversationApi.remove(id);
    if (resp.code === 0) {
      setConversations(prev => prev.filter(c => c.id !== id));
      if (currentConv?.id === id) {
        setCurrentConv(null);
      }
    }
  };

  const handleUpdateTitle = async (id: number, title: string) => {
    const resp = await conversationApi.updateTitle(id, title);
    if (resp.code === 0) {
      setConversations(prev => prev.map(c => c.id === id ? { ...c, title } : c));
      if (currentConv?.id === id) {
        setCurrentConv(prev => prev ? { ...prev, title } : null);
      }
    }
  };

  const handleSend = async () => {
    if (!inputText.trim() || !currentConv || streamState.isStreaming) return;

    const text = inputText.trim();
    setInputText('');

    // 先添加用户消息到界面
    const userMsg: ChatMessage = {
      id: Date.now(),
      role: 'user',
      content: text,
      createdAt: new Date().toISOString(),
    };
    setMessages(prev => [...prev, userMsg]);

    try {
      const result = await startStream(currentConv.sessionId, text, currentConv.agentVersion);

      // 流式完成后，将助手消息加入列表
      const assistantMsg: ChatMessage = {
        id: Date.now() + 1,
        role: 'assistant',
        content: result.content,
        createdAt: new Date().toISOString(),
      };
      setMessages(prev => [...prev, assistantMsg]);

      // 如果生成了新标题，刷新列表
      if (result.title && result.title !== currentConv.title) {
        handleUpdateTitle(currentConv.id, result.title);
      }
    } catch (e) {
      // 错误已在hook中处理，界面显示错误消息
      const errorMsg: ChatMessage = {
        id: Date.now() + 1,
        role: 'assistant',
        content: `错误：${streamState.error || '对话异常'}`,
        createdAt: new Date().toISOString(),
      };
      setMessages(prev => [...prev, errorMsg]);
    }
  };

  const handleCancel = () => {
    if (currentConv) {
      cancelStream(currentConv.sessionId);
    }
  };

  return (
    <div className="chat-layout">
      <Sidebar
        conversations={conversations}
        currentId={currentConv?.id}
        onSelect={setCurrentConv}
        onCreate={handleCreateConv}
        onDelete={handleDeleteConv}
        onUpdateTitle={handleUpdateTitle}
        agentVersion={agentVersion}
        onChangeVersion={setAgentVersion}
      />
      <div className="chat-main">
        <div className="chat-header">
          <h2>{currentConv?.title || '请选择或创建一个对话'}</h2>
          <ThemeToggle theme={theme} onToggle={onToggleTheme} />
        </div>
        <div className="message-list-container">
          <MessageList
            messages={messages}
            streamingContent={streamState.isStreaming ? streamState.content : null}
            streamingAction={streamState.action}
            theme={theme}
          />
          <div ref={messagesEndRef} />
        </div>
        <ChatInput
          value={inputText}
          onChange={setInputText}
          onSend={handleSend}
          onCancel={handleCancel}
          isStreaming={streamState.isStreaming}
          disabled={!currentConv}
        />
      </div>
    </div>
  );
};

## 5. 方案变更记录

### 变更1（2026-04-27）：每个Agent使用独立的LLM客户端和Completion

**变更原因：**
现有`AgentBeanConfig`中所有Agent共享同一个`LLMClient`和`LLMCompletion` Spring Bean，无法支持不同Agent使用不同模型（如SimpleAgent用`gpt-4o-mini`，ReActAgent用`gpt-4o`）或不同API密钥。需要在Web模块层为每个Agent独立创建LLM客户端和Completion实例。

**设计原则：**
- 不改动Agent核心层（`open-sagent-core`、`open-sagent-infrastructure`），保持Agent与Spring解耦
- 仅在Web模块的`AgentBeanConfig`中为每个Agent创建独立的`LLMClient` + `LLMCompletion`
- `ChatService`、`TitleAgentService`注入各自对应的`LLMCompletion`，创建Agent时传入

**文件变更：**

| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `open-sagent-web/src/main/java/ai/sagesource/opensagent/web/config/AgentBeanConfig.java` | 修改 | 为simple、smart、title三个Agent分别创建独立的LLMClient和LLMCompletion Bean |
| `open-sagent-web/src/main/java/ai/sagesource/opensagent/web/service/ChatService.java` | 修改 | 注入simpleCompletion和smartCompletion，createAgent时传入对应Completion |
| `open-sagent-web/src/main/java/ai/sagesource/opensagent/web/service/TitleAgentService.java` | 修改 | 注入titleCompletion，创建TitleAgent时传入 |
| `open-sagent-web/src/main/resources/application.yml` | 修改 | 每个Agent独立配置LLM参数（model、api-key、base-url） |

                .apiKey(apiKey)
                .baseUrl(baseUrl)
                .build();

        return createCompletion(client, smartModel);
    }

        LLMClient client = createLLMClient(titleApiKey, titleBaseUrl);
        return createCompletion(client, titleModel);
    }

                .enableTools(false)
                .temperature(simpleTemperature)
                .maxTokens(simpleMaxTokens)

                .maxIterations(smartMaxIterations)
                .build();

### 变更2（2026-04-27）：Prompt 配置化加载

**变更原因：**
根据 `project_design_web.md` SKILL 第8条要求，Agent 需使用 Prompt 组件从指定路径加载提示词，提示词路径和占位符值替换均需支持在配置文件中设置。现有方案中 SimpleAgent、ReActAgent、TitleAgent 的 System Prompt 均为硬编码，需利用已有 Prompt 模块组件改为配置化加载。

**设计原则：**
1. 不改动 Prompt 核心模块，复用已有 `ClasspathPromptTemplateLoader` 和 `FileSystemPromptTemplateLoader`
2. Prompt 路径和占位符变量均在 `application.yml` 中配置，支持 DotEnv 覆盖
3. 提供 classpath 默认模板文件，未配置或加载失败时 fallback 到默认内容
4. 仅在 Web 模块配置层（`AgentBeanConfig`）和 Service 层做调整

**文件变更：**

| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `open-sagent-web/src/main/java/.../web/config/PromptProperties.java` | 新增 | `application.yml` 中 `sagent.agent.*.prompt-*` 配置的绑定类 |
| `open-sagent-web/src/main/resources/prompts/simple-agent.txt` | 新增 | SimpleAgent 默认 classpath 模板 |
| `open-sagent-web/src/main/resources/prompts/smart-agent.txt` | 新增 | ReActAgent 默认 classpath 模板 |
| `open-sagent-web/src/main/resources/prompts/title-agent.txt` | 新增 | TitleAgent 默认 classpath 模板 |
| `open-sagent-web/src/main/java/.../web/config/AgentBeanConfig.java` | 修改 | 新增 `@EnableConfigurationProperties(PromptProperties.class)` 和三个 `PromptTemplate` Bean 加载逻辑 |
| `open-sagent-web/src/main/java/.../web/service/ChatService.java` | 修改 | 注入 `simplePromptTemplate` 和 `smartPromptTemplate`，删除硬编码 `DEFAULT_PROMPT` |
| `open-sagent-web/src/main/java/.../web/service/TitleAgentService.java` | 修改 | 注入 `titlePromptTemplate`，删除硬编码 `TITLE_PROMPT` |
| `open-sagent-web/src/main/resources/application.yml` | 修改 | 各 Agent 配置下新增 `prompt-path` 和 `prompt-vars` |

**配置示例（application.yml 新增部分）：**

```yaml
sagent:
  agent:
    simple:
      prompt-path: ${SIMPLE_PROMPT_PATH:prompts/simple-agent.txt}
      prompt-vars:
        role: ${SIMPLE_PROMPT_ROLE:AI助手}
    smart:
      prompt-path: ${SMART_PROMPT_PATH:prompts/smart-agent.txt}
      prompt-vars:
        role: ${SMART_PROMPT_ROLE:智能AI助手}
    title:
      prompt-path: ${TITLE_PROMPT_PATH:prompts/title-agent.txt}
      prompt-vars:
        max-length: ${TITLE_PROMPT_MAX_LENGTH:10}
```

- `prompt-path` 支持 `classpath:` 前缀、相对路径（默认 classpath）或绝对路径（文件系统）
- `prompt-vars` 为 Map 结构，键值对传入 `PromptRenderContext.of(vars)` 供 `DefaultPromptTemplate.render()` 替换 `{{key}}` 占位符
- 加载失败时自动 fallback 到硬编码默认 Prompt
